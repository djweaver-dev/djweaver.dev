Simple Python Blockchain

Description

This is a simple blockchain model using an extremely basic hashing algorithm (if you could even call it an algorithm). When instantiated, a genesis block is created and each subsequent block is linked to the last, creating a chain.

Improvements and Ideas

I want to experiment with persistent data storage using this kind of model and doing typical CRUD operations on that data.
